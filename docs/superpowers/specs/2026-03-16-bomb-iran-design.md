# Bomb Iran — 游戏设计规范

> 基于 Three.js 的街机风格 3D 轰炸游戏

## 概述

一款 45 度等角视角的经典街机轰炸机游戏。玩家驾驶战斗机在多种场景中自由飞行，通过投弹和机枪摧毁地面军事目标来完成关卡任务。游戏采用卡通风格渲染（Toon Shading），配合 Cannon-es 物理引擎实现真实的炸弹抛物线和建筑碎片效果。

---

## 技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| 构建工具 | Vite | 项目构建、开发服务器、HMR |
| 渲染引擎 | Three.js r160+ | 3D 场景渲染、卡通着色器 |
| 物理引擎 | Cannon-es | 炸弹抛物线、碎片飞散、碰撞检测 |
| 音频 | Web Audio API | 爆炸音效、背景音乐 |
| 数据存储 | LocalStorage | 进度保存、成就、排行榜 |
| 语言 | JavaScript (ES Modules) | 全部游戏逻辑 |

## 游戏视角

- **视角类型**：45 度等角倾斜视角（Isometric）
- **相机**：`OrthographicCamera` 或 `PerspectiveCamera`（低 FOV 模拟正交效果）
- **相机行为**：跟随玩家飞机，平滑移动

---

## 视觉风格

- **渲染风格**：卡通 Toon Shading
- **色彩**：鲜艳明亮，高饱和度
- **爆炸效果**：夸张的卡通爆炸，配合粒子和精灵图
- **建筑**：低多边形几何体 + 卡通材质
- **着色器**：自定义 `ToonShader`，阶梯式光照，描边效果

## 关卡系统

### 4 大章节，每章 3 小关 + 最终章含 BOSS

| 章节 | 场景 | 难度 | 地面目标 | 敌方防御 | 解锁道具 |
|------|------|------|---------|---------|---------|
| 第一章：沙漠风暴 | 🏜️ 沙漠 | ⭐ | 帐篷、坦克、雷达站 | 固定防空炮 | 🛡️ 护盾 |
| 第二章：城市突袭 | 🏙️ 城市 | ⭐⭐ | 工厂、桥梁、军事基地 | 防空炮 + 导弹 | ⚡ 加速、💥 大范围炸弹 |
| 第三章：海岸攻防 | 🌊 海岸 | ⭐⭐⭐ | 港口、舰艇、海防要塞 | 舰载防空 + 跟踪导弹 | 💥 集束炸弹 |
| 第四章：核心打击 | ⛰️ 山地 | ⭐⭐⭐⭐ | 地下掩体、发射井、核设施 | 全类型 + BOSS | 全部 |

### 关卡任务结构

每个小关包含：
1. **任务简报**（过场动画） — 展示目标位置和任务描述
2. **自由作战阶段** — 玩家自由操控完成目标
3. **关卡结算** — 基于评分指标给出 S/A/B/C 评级

### 评分系统

- **摧毁率**（40%）— 目标全摧毁 = 满分
- **用时**（30%）— 越快分越高
- **存活率**（30%）— 不掉血 = 额外加分
- **综合评级**：S（≥90%）/ A（≥75%）/ B（≥50%）/ C（<50%）

## 战斗系统

### 玩家能力

| 武器 | 类型 | 说明 |
|------|------|------|
| 💣 炸弹 | 主武器 | 抛物线下落（Cannon-es 物理模拟），爆炸产生冲击波和碎片 |
| 🔫 机枪 | 副武器 | 直线射击，用于清防空炮和小目标 |

- **生命值**：初始 3 点（可升级至 7 点）
- **武器切换**：键盘快捷键一键切换

### 道具系统

道具在空中漂浮，飞机飞过即可拾取：

| 道具 | 效果 | 持续时间 |
|------|------|---------|
| 🛡️ 护盾 | 抵挡 1 次伤害 | 被击中时消耗 |
| ⚡ 加速 | 飞行速度 ×1.5 | 5 秒 |
| 💥 大范围炸弹 | 下一发炸弹爆炸半径 ×3 | 单次使用 |
| ❤️ 生命恢复 | 恢复 1 点生命 | 即时 |

### 敌方防御

| 敌人 | 攻击方式 | 首次出现 |
|------|---------|---------|
| 🔴 固定防空炮 | 定时射击，弹道固定 | 第一章 |
| 🟠 跟踪导弹发射器 | 锁定后发射，需机动闪避 | 第二章 |
| 🔵 密集防空炮群 | 弹幕覆盖区域 | 第三章 |
| 🟣 BOSS | 特殊攻击模式 | 第四章最终关 |

### 物理系统（Cannon-es）

- **炸弹**：受重力影响的刚体，真实抛物线轨迹
- **建筑碎片**：爆炸时生成多个小刚体碎片，受冲击力飞散
- **冲击波**：爆炸点施加径向力，影响周围物体
- **碰撞检测**：炸弹/子弹 ↔ 建筑/敌人

## 音频设计

- **背景音乐**：每章不同的战斗 BGM
- **音效**：爆炸、投弹、机枪射击、被击中、道具拾取、UI 交互
- **使用 Web Audio API** 管理音频播放和音量控制

---

## 性能优化策略

- **物理简化**：仅对炸弹和碎片使用完整物理，其他实体简化处理
- **对象池**：复用子弹、碎片、粒子，避免 GC 压力
- **LOD**：远处建筑降低细节
- **实例化渲染**：相同几何体的建筑使用 `InstancedMesh`
- **像素比上限**：`Math.min(window.devicePixelRatio, 2)`
